# Team photos go here 📸

Drop your own image file into **this folder** (`public/assets/team/`), then
reference it by **file name only** in [`../../../data.js`](../../../data.js).

Example:
1. Add your picture here, e.g. `jane.jpg`
2. In `data.js`, on your block, set:  `photo: "jane.jpg",`

Supported: `.jpg`, `.jpeg`, `.png`, `.webp`, `.svg`
Tip: a square image (e.g. 400×400) looks best.

No photo? Leave the `photo` field out and an avatar is generated for you.
